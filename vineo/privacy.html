Politique de Confidentialité

Dernière mise à jour : Décembre 2025

Vineo ("nous", "notre", "l'application") s'engage à protéger votre vie privée. Cette politique explique comment nous collectons, utilisons et protégeons vos données.

En résumé : Vos données restent sur votre appareil. Nous n'avons pas accès à votre cave à vin ni à vos informations personnelles.

1. Données collectées

Données stockées localement sur votre appareil :

  • Informations sur vos vins (nom, appellation, millésime, notes, photos)
  • Vos préférences d'application
  • Historique des recommandations

Données synchronisées via iCloud (optionnel) :

  • Si vous activez la synchronisation iCloud, vos données sont stockées dans votre compte iCloud personnel
  • Nous n'avons pas accès à ces données

Données envoyées à nos serveurs :

  • Photos d'étiquettes pour l'analyse IA (traitées puis supprimées immédiatement)
  • Requêtes de recommandation (texte du plat uniquement)
  • Aucune donnée personnelle identifiable n'est conservée

2. Utilisation des données

Nous utilisons vos données uniquement pour :

  • Faire fonctionner l'application sur votre appareil
  • Analyser les étiquettes de vin via notre service IA
  • Générer des recommandations d'accords mets-vins

3. Partage des données

Nous ne vendons, ne louons et ne partageons pas vos données personnelles avec des tiers, sauf :

  • Services d'IA : Les photos d'étiquettes sont envoyées à notre API backend pour analyse. Ces images sont traitées en temps réel et ne sont pas conservées.
  • Apple (iCloud) : Si vous activez la synchronisation, vos données sont stockées dans votre compte iCloud selon les conditions d'Apple.

4. Sécurité

Nous prenons la sécurité au sérieux :

  • Communications chiffrées (HTTPS)
  • Aucun stockage de données personnelles sur nos serveurs
  • Données locales protégées par la sécurité native d'iOS

5. Vos droits

Vous pouvez à tout moment :

  • Supprimer vos données en désinstallant l'application
  • Désactiver la synchronisation iCloud dans les paramètres
  • Nous contacter pour toute question sur vos données

6. Achats intégrés

Les abonnements Premium sont gérés par Apple via l'App Store. Nous n'avons pas accès à vos informations de paiement.

7. Modifications

Nous pouvons mettre à jour cette politique occasionnellement. Les changements significatifs seront notifiés dans l'application.
